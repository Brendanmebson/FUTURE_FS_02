### Getting Started

1. Clone the repository
```
git clone https://github.com/your-username/den.git
cd den
```

2. Backend Setup
```
cd backend
npm install
```

Create a .env file:
```
MONGO_URI=your_mongodb_connection_string
PORT=5000
```

Start backend server:
```
node server.js
```

3. Frontend Setup
```
cd ../frontend
npm install
npm run dev
```

Visit http://localhost:5173 to see the project locally.
